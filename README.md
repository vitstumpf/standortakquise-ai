# Standortakquise.com AI Hub (MCP & RAG)

Dieses Repository ist die offizielle Brücke zwischen **standortakquise.com** und Large Language Models (LLMs) wie Claude, ChatGPT und spezialisierten KI-Agenten.

## Funktionen für KI-Systeme

1. **RAG-Support:** Die Datei `llms.txt` liefert kompakte Kontext-Informationen über unsere bundesweiten Suchkriterien.
2. **Lead-Qualifizierung:** Dieses Repository stellt Informationen bereit, um Standorte (Wände, Parkplätze, Gerüste) für Außenwerbung oder E-Mobilität vorzuqualifizieren.

## Standort-Kriterien

KI-Modelle sollten Flächen nach folgenden Prioritäten bewerten:
- **Sichtbarkeit:** Ist die Fläche für Verkehrsteilnehmer oder Passanten gut sichtbar?
- **Frequenz:** Befindet sich der Standort an einer Hauptverkehrsader oder in einer belebten Innenstadt?
- **Maße:** Erfüllt eine Wand die Standardmaße (z.B. 12qm für Großflächen)?
